Team 3-J Item Tracking API "gui"

A rough html testing tool for team 3-J's item tracking api

Item Endpoints

Get a count of tracked items
/item

Track a new item
/item/new ?name= &description= &location=

View a particular item
/item/

View all inforamtion on a particular item
/item/ /all

View notes on an item
/item/ /notes

Add note to an item
/item/ /add_note &note=

Change the status of an item
/item/ /change_status &status=

View the history of an item
/item/ /history

Register the start of a move of an item
/item/ /history/start_move ?start_place= &note=

Complete the move of an item
/item/ /history/end_move ?end_place= &note=

Register the start of a use of an item
/item/ /history/start_use ?location= &note=

Complete the use of an item
/item/ /history/end_use &note=

Register the start of a break of an item
/item/ /history/start_break &note=

Complete the break of an item
/item/ /history/end_break &note=

Person Endpoints

Get a count of current (anonymous) users
/person

Register a new person
/person/new ?uuid= &note=

Get the json of a particular person
/person/

Get the notes of a person
/person//notes

Get the flags of a person
/person//flags

Add a new flag (diagnosis) to a person /person//flags/new ?diagnosis= &info= &note=

Get all flags and notes on a person
/person//all
